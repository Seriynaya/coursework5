# Курсовая работа №5 Трекер привычек

## Описание:

Этот проект представляет собой работу с трекером привычек

## Использование:

1. Клонируйте репозиторий:
```
https://github.com/Seriynaya/coursework5
```


2. Установление зависимостей:
```
pip install django
```

3. Запуск проекта:
```
python manage.py
```


## Структура проекта:

1. users/ - приложение для работы с пользователями.
2. habits/ - приложение для работы с привычками.
